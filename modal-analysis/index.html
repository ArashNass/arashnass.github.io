Mode shapes

Each mode shown as its own animated sway — real eigenvectors, oscillating at that mode's own natural frequency. Shapes are scaled to a common visual amplitude; only the shape and period are physically meaningful, not the amplitude shown.

Engineering note — Idealised lumped-mass shear-building model (rigid diaphragms, no torsion, no P-Δ or soil-structure interaction). Periods and mode shapes are exact for that idealisation, but the idealisation itself is a simplification. For education only — not for structural design, code compliance, or a substitute for full 3D FE modal analysis.
